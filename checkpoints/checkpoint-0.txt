### 1.2 Install Dependencies
Create `backend/requirements.txt`:
```
agno
anthropic
fastapi
uvicorn
python-dotenv
pydantic
Install:
bashcd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
```
